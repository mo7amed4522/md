### PDF Export Instructions (macOS)
1. Quick method (VS Code): open this README, press `Cmd+Shift+P` → `Print` → save as PDF.
2. Pandoc method: install `brew install pandoc` then run:
   ```bash
   pandoc README.md -o Khaled_Mohamed_CV.pdf
   ```
3. Node method: install `npm i -g md-to-pdf` then run:
   ```bash
   md-to-pdf README.md --output Khaled_Mohamed_CV.pdf
   ```
